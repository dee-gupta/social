# Social
A simple web app for connecting friends with each other.

## Overview
"Social" is a lightweight web application designed to help friends connect, share, and interact. It supports basic user authentication, friend management, and photo-based posts with AI-generated captions. The initial version focuses on core functionality with plans for future enhancements like third-party logins.

---

## Pages

### Landing Page
- **Purpose**: Entry point for users to access the app.
- **Features**:
  - **Sign-In**: Allows existing users to log in using a username and password.
  - **Sign-Up**: Allows new users to create an account using a username and password.
- **Notes**:
  - Only username/password authentication is supported in this version.
  - Future enhancements: Add Google, Facebook, and X sign-up options.

### Homepage
- **Purpose**: Main interface for authenticated users to interact with the app.
- **Features**:
  1. **Search Bar**:
     - Search for other users on the platform by username.
     - Search for posts by keyword (e.g., description content).
  2. **Friends Section**:
     - Send a friend request to another user.
     - Accept or decline incoming friend requests.
     - View a list of friends currently online.
  3. **Post Creation**:
     - A dialog to create a post.
     - Post consists of:
       - A photo captured directly from the userâ€™s webcam (with validation to ensure successful capture).
       - A description automatically generated by an LLM API based on the photo, editable by the user before posting.
  4. **Feed**:
     - Displays a simple list of recent posts from friends.
  5. **Logout**:
     - Option to log out of the app.

## Tech Stack
- **Backend**: Node.js
  - Handles API logic, authentication, friend/post management, and session management.
- **Frontend**: React
  - Provides a dynamic and responsive UI for the landing page and homepage.
- **Database**: PostgreSQL
  - Stores user data, friend relationships, and posts.
- **Photo Captioning**: Google Cloud Vision API + LLM integration
  - Uses Cloud Vision API to analyze webcam photos and a lightweight LLM (e.g., via Google Natural Language API or a third-party service) to generate descriptive captions.
- **Cache**: Redis
  - Caches frequently accessed data (e.g., online status, recent posts) and manages user sessions for improved logout functionality.
